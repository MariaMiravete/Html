¿Qué es HTML?

¿Qué es HTML?

HTML es el lenguaje que se emplea para el desarrollo de páginas de web. Está compuesto por una seríe de etiquetas que el navegador interpreta y da forma en la pantalla. HTML dispone de etiquetas para imágenes, hipervínculos que nos permiten dirigirnos a otras páginas, saltos de línea, listas, tablas, etc.

Para escribir en html debemos saber que todas las etiquetas se abren con simbolos de mayor y menor (<*>), el asterico representa al contenido de la etiqueta que queremos abrir y además debemos saber que todas las que abres hay que cerrarlas, para poder continuar con otro tipo de etiqueta o atributo; es decir, si nosotros abrimos en el inicio de la pagina <html> cuando acabemos de escribir lo necesario dentro de la etqiueta deberemos de cerrarla así </*>

Para escribir un parrafó o empezar una linea en html lo haremos con la etiqueta p, que quedara así <p>, para pasar al parrafo siguiente lo haremos con <br> que nos saltadara a la siguiente lineam para eso lo pondremos al final del párrafo anterior y por ultimo con <hr> nos hace lo mismo que br pero este nos crea una linea separadora entre párrafo y párrafo

Formato

Formato

Para empezar explicare el formato de "negrita" con este formato lo que hacemos es resaltar la información con un negro más intenso para ello tendremos que escribir <strong>
Este párrafo contiene información en negrita y también en formato predeterminado.

El siguiente formato de enfatizado, se escribe de la siguiente manera <em>
Este párrafo tiene formato de enfatizado para pasar a formato predeterminado

El siguiente formato de "italica", se escribe de la siguiente manera <i>
Este párrafo tiene formato de cursiva "gráficamente" para pasar a formato predeterminado

El siguiente formato consiste en hacer ese tipo de letra más pequeña, se escribe de la siguiente manera <small>
Este párrafo tiene formato de letra más pequeña para pasar a formato predeterminado

También se puede subrayar el contenido que metas dentro de las etiquetas <u>
Este párrafo tiene contenido subrayado

cuando queramos poner información irrelevante u obsoleta y no la quieres borrar de la página lo pondremos asi <s>
Este párrafo tiene información obsoleta y irrelevante

En todos estos formatos cuando no queramos continuar con ellos tendremos que cerrarlos, si no la fuente seguira siendo la que le hemos dado el formato; </*>.
