1.¿Qué es HTML?

HTML es el lenguaje que se emplea para el desarrollo de páginas de web. Está compuesto por una seríe de etiquetas que el navegador interpreta y da forma en la pantalla. HTML dispone de etiquetas para imágenes, hipervínculos que nos permiten dirigirnos a otras páginas, saltos de línea, listas, tablas, etc.

Para escribir en html debemos saber que todas las etiquetas se abren con simbolos de mayor y menor (<*>), el asterico representa al contenido de la etiqueta que queremos abrir y además debemos saber que todas las que abres hay que cerrarlas, para poder continuar con otro tipo de etiqueta o atributo; es decir, si nosotros abrimos en el inicio de la pagina <html> cuando acabemos de escribir lo necesario dentro de la etqiueta deberemos de cerrarla así </*>

Para empezar al principio del archivo html hay que declarar que tipo de html que vamos a usar en la web, <DOCTYPE html> para seguir hay que abrir la etiqueta <html> que se cerrara al final del codigo.

Para escribir un parrafó o empezar una linea en html lo haremos con la etiqueta p, que quedara así <p>, para pasar al parrafo siguiente lo haremos con <br> que nos saltadara a la siguiente linea para eso lo pondremos al final del párrafo anterior y por ultimo con <hr> nos hace lo mismo que br pero este nos crea una linea separadora entre párrafo y párrafo


También tenemos que meter el titulo de la página web, el autor y charset (El elemento meta con atributo charset en un documento HTML está destinado a indicar la codificación de caracteres utilizada)

Mapa de Imágenes

Los mapas de imágenes es un nuevo planteamiento de navegación que incorpora una serie de enlaces dentro de una misma imagen. Estos enlaces son definidos por figuras geométricas y funcionan exactamente del mismo modo que los otros enlaces, Podemos utilizar estos mapas, por ejemplo, en portadas donde damos a conocer cada una de las secciones del sitio por medio de una imagen. También puede ser muy práctico en mapas geográficos donde cada ciudad, provincia o punto cualquiera representa un enlace a una página.
Debtro del mapa de imagenes tenemos que tener clara la existencia de 4 atributos:


  1. alt:  Que sirve parea indicar un texto que se muestra cuando situamos el ratón encima de el area
  2. shape:  Indica el tipo de área
  3. coords:  El grupo de valores númericos
  4. href:  Para indicar el destino del enlace
